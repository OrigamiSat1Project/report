\subsection{概要}
本節では本衛星の電源系について述べる．本衛星の電源系の概要を図\ref{3_1_power_diagram}に示す．本衛星の電源系は主に以下のコンポーネントから構成されている．
\begin{itemize}
	\item 太陽電池パネル(Solar Array Panel, SAP)
	\item バッテリ
	\item インヒビット回路
	\item 放出検知スイッチ・フライトピン
	\item CIB内電源回路
	\item 電源基板 (Electrical Power System, EPS)
	\item 伸展カメラ部電源系
\end{itemize}
